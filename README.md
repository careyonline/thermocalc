# ThermoCalc

A professional temperature conversion web application built using HTML, CSS, and JavaScript.

## Features

* Real-time temperature conversion
* Celsius, Fahrenheit, and Kelvin support
* Scientific reference points
* Interactive temperature reference table
* Responsive modern UI
* Copy conversion results with one click

## Technologies Used

* HTML5
* CSS3
* JavaScript
